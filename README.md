**Wave Function Collapse**
_Luke Courtney_

- Fully functional WFC implementation
- Per-direction or omnidirectional ruleset options
- Tile weighting
- Seed based generation
- Two demo tilesets/scenes
- Playable characters in each demo scene
- Navigational menu
